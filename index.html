Tags Básicas

Este trabalho tem como objetivo ingressar o nosso conhecimento sobre sites html, explorando resumidamente as tags iniciais.

Head: serve para armazenar metadados sobre o documento, esses que não são mostrados na página para o usuário. Ela inclui informações como o nome do site, tamanho da tela e tipo do teclado.

Body: é a parte do código onde o conteúdo da página é inserido, o corpo do documento. Nele, todo o conteúdo visível da página web será adicionado e mostrado para o usuário.

H1 ao H6: são as tags que definem um tipo de hierarquia de títulos, sendo H1 o título principal, contendo o conteúdo do site, enquanto o H2 a H6 são os subtítulos que dividem as seções para melhor organização e clareza da página. Em importância, os titúlos mudam de forma decrescente.

Strong ou B: ele deixa seu texto em negrito.

Em ou I: ele deixa seu texto em itálico.

U: ele deixa seu texto sublinhado.

Br: ele quebra a linha, o texto que estiver depois do br irá para a linha de baixo. Funcionando como o comando "Enter" do teclado.

Hr: ele cria uma linha horizontal, funcionando como um divisor de seções.

A: é a tag que direciona o usuário para um link, utilizando o atributo "href" que define seu destino.

Img: é a tag que insere uma imagem na página, utilizando o atributo "src" para definir o caminho da imagem e "alt" para fornecer um texto alternativo caso a imagem não seja carregada.


Com essa pesquisa começamos a entender como funciona o desenvolvimento de sites em html.
